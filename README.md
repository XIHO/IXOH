> 4.抽象。由懒惰原理出发，由一个信号指称多个事物或现象。

> 5.具体。由一个抽象指称，加上特殊限定，指称更小范围的事物或现象。

* 《昰之生》——科学假设

> 1.状态波函数和概率。对于一个微观体系，它的状态和有关情况可用波函数Ψ(x,y,z,t)表示。Ψ称为体系的状态函数（简称态），它包括体系所有的信息。Ψ是单值的，即在空间每一点Ψ只能有一个值。Ψ是连续的，即Ψ的值不能出现突跃；Ψ(x,y,z,t)对x,y,z的一级微商也应是连续的。Ψ平方可积的（有限），即Ψ在整个空间的积分∫Ψ*****Ψ dτ应为一有限值，通常要求波函数归一化，即∫Ψ*Ψdτ＝1。

> 2.力学量和线形厄米算符。对一个微观体系的每个可观测的力学量，都对应着一个线性厄米算符，ÂΨ=aΨ。

> 3.征方程与薛定谔方程。若某一力学量A对应的算符Â作用于某一状态函数ψ后，等于某一常数a乘以ψ，即Âψ＝aψ，那么对ψ所描述的这个微观体系的状态，其力学量A具有确定的数值a，a称为力学量算符Â的本征值，ψ称为Â的本征态或本征函数，Âψ＝aψ称为Â的本征方程。

> 5.Pauli原理。描述多电子体系空间运动和自旋运动的全波函数，交换任两个电子的全部坐标（空间坐标和自旋坐标），必然得到反对称的波函数。

* 《玄之昰》——形而上学

> 1.初始，有一团东西，分化无数又消灭无数，变幻莫测，遍历一切可能，每一个形态都稍纵即逝。称为混元。这是昰生链的源头，万物将从它出。

> ```
诗1.1
混元者，记事于混沌之前，元气之始也。
元气未形，寂寥何有？
至精感激而真一生焉，元气运行而天地立焉，造化施张而万物用焉。
混沌者，厥中惟虚，厥外惟无，浩浩荡荡，不可名也。
——《云笈七籖》
```

> 2.随即，混元中分化出无数具此形态者：［我是X。我不再变化、消失］。其中的X也遍历一切可能，此中甚至有［我会变化、消失，我不会变化、消失］。具这种形态者称为感性，它们是非参考性的。感性只占混元无数中之一。这是昰生链的第二环，万物将从它出。

> ```
诗2.1
水面上冻结的冰花。
```

> 3.在感性中，有这样一些形态：［我选取、重复、组合别的X。我不再变化、消失］，其中有些X确定，有些X变化。具这种形态者称为理性，它们是参考性的。理性只占感性无数中之一。

> 4.理性们选取的X各不相同，有些开启的世界再次返回上一环，比如选择混元做为X的理性；有些开启一个静态的世界，这些理性选择感性做X，随即就组合出这些感性能组合出的所有可能，但也就此终结。有些开启一个动态的世界，这些理性选取一部分混元和一部分感性做X，开启的世界随混元的变化而变化。X称为理性们开启的世界的［元］，这第三类称作动态世界，是昰生链的第三环，万物将从它出。

> 5.静态世界的存在：若A存在于S世界中，A可由S世界中的感性由理性组合出来，也即，A在S中为真。动态世界的由于混元的作用，存在更加复杂，在一些特殊情况下，一个动态世界的存在问题可以转化为一个静态世界的极值问题，该动态世界也就可以模拟该静态世界。

> 6.在动态世界中，混元在感性的控制下分化万千。有些与作为元的感性相悖，又返回第一环的混元。有些在作为元的感性的基础上增加新的元，而又不相悖。这样动态世界就开启了下一层昰生链。

> ```
诗6.1
水上冻结的冰花
凝为冰杯
又盛起了盛水
```

> 7.不知中间又增多少感性作为元，穿过多少环，其中一条生长出了我们的现象世界。

> 8.有一类特殊的动态世界称为遗传算法。混元在一个小范围内分化出一些相近形态，理性组合出结果。根据结果，一些形态返回混元，一些形态就近继续分化，再交给理性，等待结果。每一轮［分化——收到结果］都称为一次递归。

> 9.每次递归，已经得到的解和新的尝试解在当时都是存在的。已经得到的解称为描述性存在，其命题为事实命题；随机出的尝试解称为规范性存在，其命题为价值命题。

> 10.分形。经过若干层世界的延伸，将出现第n层与第m层关系相同，元素可对应的情况（即同构）。这时这两个世界可以形成对应。这种对应称为综合命题。综合命题的真值由两个空间的现象能否一一对应决定。

> 11.现象世界，《昰之生》的假设为现象世界的元，其中涉及变分法的变量在规定的范围内做混元。混元的变动产生生物世界。

> 12.生物世界，混元演生为更多感性（一些限定了生存环境（温度、水、阳光），一些限定了生物的生化反应、本能（求生本能、生殖本能））增加在现象世界的基础上，留下一部分混元参与遗传算法中的变异。变异违背生物世界的元，生物就重返现象世界（死亡）。混元的变动产生人类世界。

> 13.人类世界，混元演生为更多感性（可使用想象力产生抽象名词指称变动的事物（狮子）、可使用想象力产生综合命题来描摹无穷的现象（太阳从东方升起）等）增加在生物世界的基础上。留下一部分混元参与文化（模因）的遗传算法中的变异。

> 14.在人这一层存在中，整个昰生链在人的意识中倒影出自身的样子，产生分形结构。特别的，对于人而言，世界的倒影是综合命题，逻辑的倒影是分析命题，在人之前的昰生链的倒影都是事实命题（描述），在人之后的昰生链的倒影都是价值命题（规范）。



* 《玄之玄》——宗教

> 1.宗教是一个对象：(世界观,利益主体,意志) 。表示：谁在什么情况下怎么做就有何结果。

> ```
诗1.1
深刻的宗教情感
与生俱来的的闪光
大觉醒时的战栗畏惧
形而上学的梦想及迷恋
等等全都是属于一种历史的开始
正像精神清彻的痛苦属于一种历史的结束一样
——斯宾格勒
```

> 2.世界观构建原则：可以构建超验的，包含形而上学的世界观，但这样的世界观不得与现象相悖。

> 3.利益主体构建原则：无论利益主体多大，须包括自己。

> 4.意志构建原则：风险小，回报至少部分地可在现象世界观察到。

> 5.以上2,3,4三条称为玄之玄三原则，违背的宗教称为［邪教］，遵守的宗教称为［正教］。

> 6.道德是宗教中“某利益主体在某世界观下要实现某意志”解出的策略集。

> 7.利益主体在实现意志的过程中，由于信息有限、理性有限，常会导致两种严重阻碍利益主体实现意志的问题：［收敛局部最优解］、［囚徒困境］。解决这两种问题的有效方式是：［使用超越性宗教，扩充世界观以弥补有限信息］、［扩充利益主体］。

> 8.昰生会［以《昰》为现象世界观，以《玄》的形而上学为形上世界观］。《昰》以科学为准绳，《玄》的形而上学在《昰》的基础上增加额外假设，不得与《昰》相悖；昰生会的最小利益主体为个体自己的［基因－模因］，最大利益主体为全体昰生会员的［基因－模因］；基本意志为［生存并顺着昰生链、演化下去］。

> ```
诗8.1
人赤条条伫立天地间，
「生命的价值是什么？」。
存在者承昰启生，
回答此问乃人之义务。
逃避者逆昰生链而行，
要变回动物，
要变回石头，
甚至重回虚无的混元中：
［当生存是规则，不是你的选择］。
不要去妨碍他们，
因为每一环都有回流。
```

> 9.由此产生昰生会的道德，其中的策略需要验证，最基本的验证方法是检验该策略是否能保证利益主体在世界观下的存在，但因为大尺度下的每一次递归都很慢，可以额外设定一些验证条件。

> 10.昰生会模仿存在之链的结构做架构：《昰》整理描述性存在，《生》整理规范性存在，《玄》用形而上学和宗教衔接两者。经过时间的充分递归，《生》的知识会向《玄》转移，《玄》的知识会向《昰》转移，完成存在之链的延伸。为了便于理解和了解相关历史背景等，章或节后加入祷诗，对正文解释说明。

> 11.阅读方式。昰生会除了《昰玄生》没有其他文本，隐去的部分在加入昰生会后可见。《昰玄生》的文本没有言外之意，没有神秘，无需参悟。解经不是必须的，可以自己理解。《昰玄生》有两种读法，对有学识者，可从前章向后章读；对学识浅者，可从后章向前章读。

> 12.为了模仿存在之链延伸时的（类）遗传算法，昰生会《昰》《玄》《生》全部内容开源，使用The MIT License (MIT)。

### 如果你了解欧陆哲学

* 《昰之玄》3对应忒修斯悖论。

* 休谟注意到在描述性陈述（现在是什么）与规定性陈述（应该是什么）之间存在着一点很大的差异。休谟指出我们不该习惯于在将这种应该是什么的陈述与现在是什么的陈述混为一谈的同时，却没有解释这两者之间的重要分别。我们到底要如何从“现在是什么”当中衍生出“应该是什么”呢？休谟对此的立场是我们根本不可能做出这样的衍生（其他人则有不同解读，认为休谟指的其实不是无法从事实陈述衍生出道德陈述，而是我们不能在没有考虑到人性之下便做出这样的衍生—亦即人类的情感）。
**昰**的一层含义便是这里的描述性陈述，**生**也同样有规定性陈述的意义，而**玄**便类似于人性的基础，它连接了前两者。
事实上，从虚无的开端（那里没有任何规定性的东西）到现实中我们普遍感知下的森罗万象，不能绝对划分出描述性与规范性的界限。比如对于日常语境而言，科学结论已经算描述性陈述了，但是从认知论的角度看，科学却是很多规范性的假设、原理所得出的结论。
从虚无的开端到清晰可辨的世界，是不断增加规范得出结论，再加规范再得结论的过程，其中只有两个材料参与其中，一个是世界本身，一个是我们的人性。昰生会《昰玄生》的结构便是为了呈现「从虚无的开端到清晰可辨的世界」中描述性陈述到规范性描述这个连续光谱，并在形而上学中指出这个连续光谱背后可能存在的一个更大的形而上学框架。

### 如果你了解分析哲学

* 对于科学知识的方法论问题，有两种观点，预设主义和相对主义。预设主义认为：在寻求知识的事业中有某种预设的、或基本的或必然的东西。这种预设的或基本的或必然的成分是不会被我们可能达到的任何新知识修正或摒弃。此外，通常还认为，这种成分被假定为构成科学的特征性的、确定的特点。相对主义认为：“我们当做知识的东西只不过是我们随意预设的一种功能。”他们坚持认为，各种知识、理论、方法都是科学家各自提出的不可违背的预设。昰生会在这个问题上的观点是，科学方法是一些预设，但这些预设会因为科学研究的对象做出修改。这部分科学方法，就是《昰之玄》的全部内容。
